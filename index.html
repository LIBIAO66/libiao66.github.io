购物网站的登陆注册模块

登陆 在 html页面中 像表单中的密码，用户名，邮箱，出生日期等用的都是正则验证，用户要和数据库中的用户进行对比，所以采用的是ajax验证，
当表单中所有数据符合规范时就可以提交数据了表单提交后，servlet页面中获取表单数据然后保存在对象数组中，然后写dao方法，如果insert成功则返回值大于0，
则重定向到登陆页面，这边我用javascript写了一个五秒自动跳转。登陆的话就是你表单中提交的数据与数据库中数据进行对比，用户名密码正确的话就跳转到主页面上，
将登陆人的姓名放在session中，错误的话就再次返回到登陆页面并对其错误信息进行提示

我还写了后台分页展示数据库中数据。
//设置每页显示用户 的多少
int pageSize=5;
//获取当前页码
String Num= req.getParameter(“pageNum”);
//带分页查询用户信息的实现方法
//调用带有分页的实现方法
Page page =SupManagerDao.queryAllCustom(pageSize, Num);
session.setAttribute(“Page”, page);
session.setAttribute(“Servlet”, “servlet/SuperManagerPersonServlet?action=queryAllCustom&pageNum=”);
//去显示顾客信息
